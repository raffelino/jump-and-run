# Level Generator ausführen

## Schnellstart

```bash
# Level generieren
node generate-levels.js

# Oder mit npm
npm run generate-levels
```

## Was passiert?

Das Script generiert alle 5 Welten mit jeweils 5 Levels und speichert sie in:

- `js/levels/world1.js` - Grasland
- `js/levels/world2.js` - Dunkle Höhlen  
- `js/levels/world3.js` - Brennende Wüste
- `js/levels/world4.js` - Eisige Berge
- `js/levels/world5.js` - Himmelsburg

## Nach der Generierung

1. Öffne die generierten Dateien in deinem Editor
2. Bearbeite die Level nach deinen Wünschen
3. Speichere die Änderungen
4. Lade das Spiel im Browser neu

## Wichtig

⚠️ **ACHTUNG**: Das erneute Ausführen überschreibt alle world*.js Dateien!

Sichere deine manuell editierten Level, bevor du neu generierst!
